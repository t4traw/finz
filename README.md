アングラーのタックルボックスなど、釣具や装備写真を共有するサービス

## Requirements

- ruby(2.6.3)
- nodejs(v14.4.0)
- overmind

## Development

### Setup

```
$ bin/setup
$ npm run dev
```

### Running test

```
$ npm run test
```
